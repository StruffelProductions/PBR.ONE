PBR.ONE is an easy to use toolkit for creating interactive previews of digital materials for Physically Based Rendering and HDR panoramas for use by CG- & VFX artists.

Visit on GitHub

Flexible, Dynamic and Accurate.

All previews are completely interactive - giving users a quick yet detailed look at any assets before downloading.

Embed & Link Anywhere.

Let's face it: Most artists don't want to code and maintain their own website and use third-party services and stores instead. This is why PBR.ONE is build to be easily linked or embeded into any other site using an iframe.

How it works:

All you need to to get started is to upload your material maps or HDRIs somewhere on the internet (more about that in the documentation) and point PBR.ONE at them:

<iframe src="https://cdn.pbr.one/0.3.0/texture-tiling.html#texture_url=https://pbr.one/media/bricks_col.jpg">

...and your preview is done. This example is just a tiled preview of a single map but you can do quite a lot more! Check the documentation for more details and for places to easily upload files for the online previw.
