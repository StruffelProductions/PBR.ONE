material-shading.html - Basic Example

This example shows a single PBR material of a brick wall with multiple maps referenced via URL. Additional parameters like the tiling scale are set as well. At the moment the clay render mode is still enabled. Try changing clayrender_enable to 0.

hdri-exposure.html - Basic Example

This view can be used to see the full dynamic range of an HDRI panorama. The environment_exposure is currently set to a very dark -6 EVs. You can adjust the value in the text field or using the slider.

hdri-shading.html - Basic Example

This view allows displays the HDRI as an actual environment with several dummy objects placed in the center to see the shading and reflections. These preview objects are currenly disabled. You can enable them by setting spheres_enable to 1 in the text field.

texture-tiling.html - Basic Example

This view simply creates a tiled preview of a texture.

material-shading.html - Advanced Example

Multiple PBR materials can be loaded by specifying multiple maps for every channel. You can use the "Texture" dropdown to switch to the other material and the "Environment" dropdown to change the environment.

hdri-exposure.html - Advanced Example

This view can be used to see the full dynamic range of an HDRI panorama. In this example two HDRIs are loaded which can be selected using the "Environment" dropdown.

hdri-shading.html - Advanced Example

Like all the others this view also supports multiple HDRIs.

texture-tiling.html - Advanced Example

More than one texture can be loaded in this view to show off all the maps of a PBR material.
